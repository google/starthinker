DV360 Bulk Editor

Allows bulk editing DV360 through Sheets and BigQuery.

Instructions

Select Load Partners, save the recipe, then run. A Sheet called DV_Editor_ will be created.
In the 'Partners' sheet tab, fill in Partners tab filter.
Select Load Advertisers, save the recipe, then run.
In the 'Advertisers' sheet tab, fill in Advertisers tab filter.
Select Load Campaigns, save the recipe, then run. Filtering by campaigns is optional.
Now, select Load Insertion Orders and Line Items, save the recipe, then run.
To patch, fill in changes on all tabs with colored fields.
Select Preview, save the recipe, then run the recipe.
Check the Audit and Preview tabs.
To Patch, select Patch, save the recipe, then run.
Check the Success and Error tabs.
Patch can be run multiple times.
Patch ONLY changes edited fields.
